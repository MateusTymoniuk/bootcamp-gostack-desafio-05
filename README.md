## 🚀 **Sobre**
Neste desafio, foi construída uma aplicação que busca os repósitorios do github (digitado pelo usuário). É possível também, detalhar o repósitório, juntamente com suas issues.

O desafio consistia em adicionar quatro funcionalidades à aplicação:
- Captar erros, caso o usuário digitasse um repositório que não existisse
- Não permitir a entrada de repositórios duplicados
- Adicionar um filtro de estado às issues
- Adicionar paginação à lista de issues

Desafio proposto em: https://github.com/Rocketseat/bootcamp-gostack-desafio-05/blob/master/README.md

## 🚀 **Instalação**
1 - Clonar o <a href="https://github.com/MateusTymoniuk/bootcamp-gostack-desafio-05">repositório</a> em seu computador;

2 - **Instalar as dependências do projeto** digitando no terminal o comando:

    npm install

3 - Para **executar a aplicação** utilize:

    npm start

ou, no caso de construir a aplicação para gerar a build final:

    npm run build
